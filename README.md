# VERSIA 
Red social universitaria.

## Requisitos
- NodeJS
- MariaDB

## Instalación
1. Ejecutar script de creación de base de datos
2. `npm install`
3. `node app.js`
4. [Ir a Versia](http://localhost:8080) 

## Tecnologías y Herramientas
Front-end
- Bootstrap 4
- JQuery

Back-end
- NodeJS + ExpressJS
- MariaDB
